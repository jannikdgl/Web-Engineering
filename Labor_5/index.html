HTML

HTML (Hypertext Markup Language) ist die Basis für die Strukturierung von Inhalten im World Wide Web. Es handelt sich um eine Auszeichnungssprache, die aus verschiedenen Elementen besteht. Diese Elemente definieren die verschiedenen Teile einer Webseite, wie Überschriften, Absätze, Bilder, Links und vieles mehr. HTML ermöglicht die semantische Strukturierung von Webseiten, was bedeutet, dass der Code nicht nur das Aussehen, sondern auch die Bedeutung der Inhalte beschreibt.

Begriffe

Elemente: Bausteine in HTML, z.B. für Absätze oder für Bilder.

Tags: Markierungen, die den Anfang und das Ende eines HTML-Elements kennzeichnen.

Javascript

JavaScript ist eine Skriptsprache, die in Webbrowsern ausgeführt wird und es ermöglicht, Webseiten dynamisch zu gestalten. Mit JavaScript können Benutzerinteraktionen verarbeitet, Inhalte geändert und sogar serverseitige Anfragen gestellt werden. Diese Sprache spielt eine entscheidende Rolle bei der Entwicklung moderner, interaktiver Webanwendungen.

Begriffe

DOM (Document Object Model): Eine Baumstruktur, die die hierarchische Repräsentation von HTML-Dokumenten im Browser darstellt.

Event: Ein Ereignis, wie das Klicken auf einen Button oder das Laden einer Seite, das von JavaScript abgefangen und verarbeitet werden kann.

CSS

Cascading Style Sheets (CSS) ist eine entscheidende Technologie im Webdesign, die die Darstellung von HTML-Elementen auf einer Webseite steuert. Durch die Trennung von Inhalt und Design ermöglicht CSS eine effiziente Gestaltung und Anpassung von Webseiten. CSS wird in allen modernen Browsern unterstützt und ist eine wichtige Technologie für die Entwicklung von Webanwendungen.

Begriffe

Selektoren: Selektoren sind in CSS Muster, die bestimmte HTML-Elemente auf der Seite auswählen. Sie legen fest, welche Elemente die definierten Stileigenschaften erhalten. Deklarationen: Deklarationen bestehen aus Eigenschaften und Werten, die einem ausgewählten HTML-Element zugeordnet sind. Sie definieren das Aussehen des Elements.

NodeJs

Node.js ist eine serverseitige JavaScript-Laufzeitumgebung, die auf Chrome's V8 JavaScript-Engine basiert. Sie wird verwendet, um skalierbare Netzwerkanwendungen zu erstellen. Node.js kann auf verschiedenen Betriebssystemen ausgeführt werden und ist eine Open-Source-Plattform. Sie ermöglicht es Entwicklern, JavaScript für die serverseitige Skriptausführung zu verwenden, um dynamische Webseiten zu erstellen.

Begriffe

Node.js: Eine serverseitige JavaScript-Laufzeitumgebung, die auf Chrome's V8 JavaScript-Engine basiert.

V8: Eine Open-Source-JavaScript-Engine, die in Google Chrome und in Node.js verwendet wird.
